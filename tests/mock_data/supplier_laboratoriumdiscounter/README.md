# Laboratorium Discounter Mock Data

The mock data was generated by using the below bash command:

```bash
#!/usr/bin/env bash 

query="${1:-acid}"
curl -s --get "https://www.laboratoriumdiscounter.nl/en/search/${query}/" \
    --data "format=json" \
    --data "limit=200" |
    jq --raw-output
```
